# 🎰 FORTUNE FIRE 〜200円から世界の頂点へ〜

ブラウザだけで動く、激熱演出の **デジパチ風パチンコゲーム**。
ビルド不要・依存ゼロ。`index.html` を開くだけで遊べます。音は全て Web Audio API で合成（外部音源なし）。

![type](https://img.shields.io/badge/type-pachinko-ff6ec7) ![tech](https://img.shields.io/badge/tech-HTML5%20Canvas%20%2B%20WebAudio-3af0ff) ![build](https://img.shields.io/badge/build-none-success)

## 遊び方
1. `index.html` をブラウザで開く（またはローカルサーバで配信）。
2. **「発射」ボタンを長押し**して玉を打ち出す。始動口に入ると保留が貯まり、デジタル図柄が回転します。
3. 図柄が3つ揃えば **大当り**！ 確変図柄なら **確変（連チャンモード）** に突入。
4. 「オート」で自動打ち出し、「スペック」で台を変更、「デバッグ ▲」から各演出を強制再生できます。

> 初回は音が鳴るようにボタンを一度タップしてください（ブラウザの自動再生制限）。

## 実機準拠UI
- 台上部の**データカウンター**（大当り回数・確変回数・最高連・回転数・ハマり・合成確率／7セグ表示）と**スランプグラフ**（差玉・大当りマーカー）
- デジタル**図柄数字**（奇数=確変/金、偶数=通常/白）、キャビネット枠＋ランプバー、電サポ中の**右打ち**＆RUSH背景
- 大当り中の**ラウンド/出玉カウンターパネル**、光沢保留＋変動中ランプ＋先読み昇格
- **スマホ完全対応**（320/375/390px で横はみ出しゼロを端末エミュレーションで検証）

## 熱い演出と信頼度
演出は「当落を先に確定 → 結果に応じて重み付き抽選」する設計のため、各演出の信頼度
`P(当り|演出)` が**理論計算でき、モンテカルロで検証済み**です（`test/sim.mjs`）。

- **保留先読み**：白 → 青(5%) → 緑(15%) → 赤(40%) → 金(80%) → 虹(100%)
- **ステップアップ予告**：STEP2(7%) → 3(25%) → 4(55%) → 5(92%)
- **リーチ**：ノーマル(7%) → スーパー/ドヤ(30%) → 激熱カットイン/黒塗り(70%) → 金/ヒーロー(90%) → 全回転/50年後(98%)
- **擬似連**・**復活演出**・**確変連チャン** など

信頼度は通常時(1/319)基準の設計値。低スペックほど同じ演出の信頼度が自然に上がります（実機と同じ挙動）。
